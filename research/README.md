# V2 strategy research

폐기된 V1 라이브 봇의 종목별 추천값이나 점수 체계를 사용하지 않고, 설명 가능한 단순 전략군을 같은 조건에서 비교하기 위한 V2 독립 연구 도구다.

## 2026-07-15 예비 백테스트

### 조건

- 유니버스: 2026-07-15 현재 S&P 100 구성 종목 스냅샷. Alphabet은 GOOGL만 남겨 주식 종류 중복 진입을 제거했다.
- 데이터: Yahoo Finance 수정 일봉, 2017-05-01~2026-07-13.
- 체결: 당일 종가로 신호를 계산하고 다음 거래일 수정 시가에 체결한다.
- 비용: 매수와 매도 각각 0.1%. 민감도 검사는 각각 0.2%로 재실행했다.
- 공통 한도: 종목당 10%, 총노출 60%, 업종 20%, 동시 보유 8종목, 거래당 계획위험 0.5%.
- 공통 신규매수 조건: SPY 종가가 SPY SMA200 위에 있을 때만 허용한다.
- 공통 손절: 진입 시점 2.5 ATR을 사용하되 4~10% 범위로 제한한다.
- 현금 이자, 실적발표 필터, 세금은 포함하지 않았다.

S&P 100은 실제로 시점에 따라 구성 종목이 달라지지만 무료 공개 자료만으로 장기간 일별 편입 이력을 완전하게 복원하지 못했다. 따라서 이 결과에는 **현재 살아남은 우량주를 과거에도 보유 후보로 본 생존편향**이 있으며, 실전 전략 확정 근거가 아니라 후보 제거와 후속 연구 순서 결정에만 사용한다.

검증 구간은 2021~2023년, 홀드아웃은 2024-01-02 이후다. 이 보고서를 작성하면서 홀드아웃 결과를 이미 확인했으므로 같은 구간을 이후 전략 수정의 최종 미사용 검증구간으로 다시 사용하면 안 된다.

### 생존 후보 기준

기본, 비용 2배, 장기 이력 제한 시나리오에서 전체·개발·검증·홀드아웃 CAGR이 모두 0%를 초과해야 활성 후보로 남긴다. 개별 연도의 손실까지 모두 금지하면 현실적인 전략도 제거될 수 있으므로, 사전에 나눈 주요 검증 구간을 판정 단위로 사용한다.

- `RSI(2) 반등 + SMA200`: 전체와 모든 주요 구간이 음수여서 탈락했다. 승률 54.76%는 손익비 악화를 보상하지 못했다.
- `볼린저 하단 재진입 + SMA200`: 승률은 60.43%지만 검증 CAGR이 -0.91%이고 비용 2배 전체 CAGR도 -0.78%라 탈락했다.
- 탈락 전략은 기본 `active` 실행에서 제외한다. 비교 재현이 필요할 때만 `--strategy-set archive`로 실행한다.

날짜가 고정된 최초 결과 JSON은 선별 전에 생성되어 `strategySet: core`와 여섯 후보를 담고 있다. 원본 결과를 사후 수정하지 않고 증거로 보존하며, 선별 이후 실행부터 `active`는 네 생존 후보, `archive`는 최초 여섯 후보를 뜻한다.

양수 수익 필터 통과만으로 경제성이 입증되는 것은 아니다. 같은 기본 구간의 `SPY 60% + 현금 40%` CAGR은 10.84%였고, 생존 후보 네 개의 CAGR은 3.27~10.64%였다. 상대 모멘텀은 더 낮은 최대낙폭으로 벤치마크에 근접했지만 생존편향 우려가 크며, 나머지 후보는 낮은 낙폭이 낮은 절대수익을 보상하는지 다음 단계에서 비교해야 한다.

### 비용과 유니버스 민감도

- 편도 비용을 0.2%로 두 배 올려도 EMA20/100은 CAGR 4.17%, 최대낙폭 10.10%, Profit Factor 1.491이었다.
- Donchian 55/20은 같은 조건에서 CAGR 3.51%, 최대낙폭 15.47%, Profit Factor 1.323이었다.
- RSI(2)는 비용 2배에서 CAGR -5.96%로 더 악화됐고 볼린저는 CAGR -0.78%로 음수가 됐다.
- 2017년 이전 가격 이력이 없는 GEV, HONA, PLTR, UBER를 제외하자 상대 모멘텀 CAGR은 10.64%에서 8.50%, 홀드아웃 CAGR은 23.79%에서 16.33%로 크게 낮아졌다. 현재 구성 종목 생존편향에 민감하다는 경고다.
- 같은 장기 이력 제한에서 EMA20/100은 CAGR 5.16%, 최대낙폭 9.18%로 거의 변하지 않았다.

EMA20/100과 EMA50/200이 비슷한 수익과 낙폭을 보여 EMA 추세 계열은 특정 주기 하나에만 의존하지 않았다. Donchian은 100/40에서 개선됐지만 인접 주기의 낙폭과 성과 차이가 더 커 추가 검증이 필요하다.

### 현재 판단

1. **활성 생존 후보는 RSI(14), EMA20/100, Donchian 55/20, 상대 모멘텀 네 개다.** 이 판정은 양수 수익 필터를 통과했다는 뜻이지 실전 승인을 뜻하지 않는다.
2. 실전 채택 후보 1순위는 아직 전략 하나가 아니라 EMA 추세 계열이다. 로우리스크 우선이면 EMA50/200, 신호를 조금 더 자주 받고 싶다면 EMA20/100이 다음 비교 대상이다.
3. Donchian 100/40은 수익 후보로 남기되, 한 주기에서 유난히 좋아진 결과인지 더 넓은 시점별 유니버스에서 확인해야 한다.
4. 상대 모멘텀은 절대 수익은 가장 높지만 검증구간 정체, 최근 홀드아웃 집중, 현재 구성 종목 편향 때문에 연구 전용으로 유지한다.
5. RSI(14)는 모든 구간이 양수지만 전체 CAGR 3.27%로 낮다. 현금성 수익률과 세금을 포함한 다음 비교에서 경제적 의미가 있는지 다시 판정한다.
6. 높은 승률 평균회귀 가설은 이번 조건에서 지지되지 않았다. 가장 높은 승률인 볼린저도 검증구간과 비용 민감도에서 음수였다.
7. 이 결과는 `작게 자주 이기는 전략`보다 `자주 지더라도 평균 이익이 평균 손실보다 큰 단순 추세 전략`이 더 견고할 수 있음을 보여준다. 다만 과거 구성 종목 데이터와 새로운 미사용 구간 없이는 실전 확정하지 않는다.

## 실행

```bash
npm run research:v2
npm run research:v2:verify
```

탈락 전략까지 포함한 최초 여섯 후보를 다시 비교하려면 보관 실행을 사용한다.

```bash
npm run research:v2:archive
```

직접 민감도 검사를 실행하려면 다음 옵션을 사용한다.

```bash
node research/v2-strategy-backtest.mjs \
  --strategy-set sensitivity \
  --cost-per-side 0.002 \
  --require-history-before 2017-01-01 \
  --output research/results/custom.json
```

원자료는 [기본 결과](results/v2-strategy-backtest-2026-07-15.json), [비용 2배](results/v2-strategy-backtest-2026-07-15-double-cost.json), [장기 이력 제한](results/v2-strategy-backtest-2026-07-15-long-history.json), [주기 민감도](results/v2-strategy-sensitivity-2026-07-15.json)에 있다.
