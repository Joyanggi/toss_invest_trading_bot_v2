# WORKLOG

### 2026-07-15 - 작성자: Codex
- 문제/목적: 보조지표라는 사전 제약 없이 S&P 500 100% 매수보유를 장기적으로 이길 수 있는 자동매매 후보를 설계하고 엄격한 기준으로 선별한다.
- 진단(근거): 기존 연구는 총노출 60%와 현재 S&P 100 구성 종목을 사용해 절대수익 벤치마크 초과라는 새 목표와 맞지 않았다. 강세장에서 현금을 많이 보유하는 구조는 SPY를 이기기 어렵고, 현재 구성 종목을 과거에 적용하면 생존편향이 생긴다.
- 수정 내용: 장기 상장 ETF를 사용하는 독립 월간 백테스트 엔진을 추가했다. 섹터·스타일·듀얼 모멘텀, SPY 추세·변동성 관리, 앙상블 여섯 후보를 결과 확인 전에 고정했고 SPY 100%와 동일한 다음날 시가·거래비용 조건으로 비교했다. 개발·검증·홀드아웃, 비용 2배, 5년 구간 승률을 모두 통과해야 하는 자동 선별 기준을 추가했다.
- 건드린 파일/함수: `research/benchmark-challenger-backtest.mjs`, `research/verify-benchmark-challenger.mjs`, `research/results/benchmark-challenger-2026-07-15.json`, `research/BENCHMARK_CHALLENGER.md`, `README.md`, `research/README.md`, `package.json`.
- 검증 방법과 결과: 19개 ETF, 2008-07-01~2026-07-13의 4,535거래일을 수집했고 실패는 0건이었다. 기본·비용 2배 시나리오와 결과 구조 검증이 통과했다. SPY 첫 수정시가와 마지막 수정종가의 단순 매수보유 계산이 엔진 최종자산과 2달러 이내로 일치했다. 사전 기준 생존 후보는 0개다.
- 가정/주의(다른 에이전트가 검토해야 할 점): 듀얼 모멘텀은 전체 CAGR이 SPY보다 1.28%p 높았지만 홀드아웃에서 -3.36%p, 5년 승률 52.23%로 탈락했다. 개별 종목 구성 생존편향은 피했으나 현재 생존 ETF를 선택한 편향은 남는다. 이번 홀드아웃을 확인했으므로 같은 구간을 보며 파라미터를 조정하지 않는다.
